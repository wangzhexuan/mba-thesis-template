# MBA论文撰写AI辅助工具

> 一套"陪你从开题写到答辩"的 AI 辅助写作工具包：7 阶段工作流、即用 Prompt、文档模板与配置。
> **你负责拿主意，AI 帮你查资料、写初稿、改格式、降重、练答辩。** 全程不用懂编程。

## 这是什么

把"写一篇 MBA 论文"拆成从开题到答辩的几十个可执行小步骤，每一步都配好了"该对 AI 说哪句话"。你只需复制提示词、填上自己的信息、发给 AI。适用于 **MBA / EMBA / DBA 等管理类学位论文**，域中立——营销、绩效、供应链、运营等选题都能用。

## 快速开始

```bash
# 1. 复制一份模板，原件留着下次用
cp -r mba-thesis-template 我的MBA论文 && cd 我的MBA论文

# 2. 建项目：双击 init.html 填表，或让 AI 读 AGENTS.md 帮你建
./init.sh

# 3. 打开第一份手册，从第 0 关开始照着做
open workflows/00-setup.md
```

不想用命令行也行——详见《操作指南》，里面有最省事的网页向导用法。

## 适合谁

- 正在写 MBA / EMBA / DBA 或其他管理类学位论文的同学；
- 偏"分析一家公司、用管理理论解决实际问题"的应用型选题（也支持问卷调研型）；
- 不用懂编程，会用电脑、会复制粘贴、能上知网即可。
