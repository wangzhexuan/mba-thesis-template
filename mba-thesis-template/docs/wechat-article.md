## 一、MBA论文的痛，每个MBAer都懂

白天上班，晚上带娃，周末还要写论文。

开题报告改了8遍，导师说"问题不够聚焦"。文献综述写了6000字，被批"像流水账"。好不容易写完初稿，查重率38%，降重降到怀疑人生。

更扎心的是：你知道要用AI帮忙，但每次跟AI对话都像对牛弹琴——它给你的东西要么太泛、要么太学术、要么跟你的情况完全不搭。

**问题不在AI，在你跟AI之间的"翻译层"。**

这篇文章介绍一套我亲测有效的MBA论文AI辅助写作工作流。它不是让你把论文交给AI写，而是帮你**把论文拆解成AI能理解的小任务**，一步步推进，每一步都有具体的Prompt可以复制使用。

---

## 二、核心理念：你指挥，AI干活

这套工作流有一个核心原则：

> **你负责决策，AI负责执行。**

具体来说：

- **你决定**：选什么题、用什么理论、写什么策略、论文最终长什么样
- **AI帮你**：检索文献、整理资料、生成初稿、检查逻辑、格式排版、降重润色

AI是你的写作助手，不是你的枪手。你需要对每一个重要决策把关确认。

---

## 三、7个阶段，从0到答辩

整个工作流分为7个阶段，按顺序推进：

```
Phase 0  项目初始化     → 配置论文基本信息、学校格式、研究框架
Phase 1  文献检索与管理  → 检索、筛选、精读、撰写文献综述
Phase 2  框架设计       → 选理论、画框架图、写第1章
Phase 3  数据收集       → 设计问卷/访谈、收集数据、初步分析
Phase 4  内容生成       → 写全部章节、做图表、整理参考文献
Phase 5  质量审校       → 8维度评审、逻辑检查、策略可执行性检查
Phase 6  答辩准备       → 做PPT、写演讲稿、模拟Q&A
```

每个阶段都有：
- **明确的目标**：这一步要产出什么
- **具体的步骤**：怎么做，分几步
- **即用的Prompt**：直接复制，填上你的信息就能用
- **检查清单**：怎么确认这一步做完了

## 四、怎么用？三个关键

### 1. 不用记任何命令

你不需要知道什么是`/paper-lookup`、`/scientific-writing`。

你只需要用**自然语言**告诉AI你要什么：

```
❌ 旧方式：/paper-lookup "你的研究主题"
✅ 新方式：帮我检索关于你的研究主题的中文学术文献，
          优先核心期刊，近5年，返回10篇。
```

AI会自动判断需要调用哪些工具，你只管提需求。

### 2. 每一步都有Prompt模板

工作流的每个步骤都提供了可以直接复制的Prompt。你只需要把`{占位符}`替换成你自己的信息：

```
请帮我撰写MBA论文中文摘要（500-800字）。
论文题目：{你的题目}
研究对象：{你的企业}
研究方法：{你的方法}
主要发现：{你的发现1}、{发现2}、{发现3}
主要结论：{你的结论}
```

**7个阶段，75个Prompt，覆盖从开题到答辩的所有环节。**

### 3. 先初稿，后打磨

不要追求一步到位。工作流的节奏是：

```
先完成 → 再完美
先有框架 → 再填内容
先写初稿 → 再用AI润色降重
```

初稿可以粗糙，重要的是把逻辑跑通。然后用AI帮你打磨语言、检查格式、降低查重率。

## 五、两种研究类型都支持

不管你的论文是**案例研究型**还是**问卷调查型**，这套工作流都覆盖了：

| 类型 | 特点 | 工作流适配 |
|------|------|-----------|
| 案例研究型 | 深入分析一个企业，用理论诊断问题、设计方案 | 第3章重现状诊断，第4章重策略设计 |
| 问卷调查型 | 大样本量化，假设检验，统计分析 | 第3章重研究设计，第4章重数据分析 |

配置文件里选好研究类型，工作流会自动适配差异部分。

## 八、常见疑问

**Q: 用AI写论文算学术不端吗？**

A: 关键在于你怎么用。AI是工具，就像用Word排版、用Excel做统计一样。最终的决策、判断、观点必须是你自己的。工作流的设计就是"你指挥，AI干活"——每个重要节点都需要你确认。

**Q: AI生成的内容质量行吗？**

A: 不要直接用AI的输出。把它当初稿，在上面改。AI擅长的是：帮你快速起一个框架、帮你整理资料、帮你检查格式和逻辑。最终的质量取决于你的修改和判断。

**Q: 需要付费吗？**

A: Claude Code有免费额度，完成一篇论文基本够用。如果用量大，Pro版每月20美元。其他工具（CNKI、Zotero等）学校一般都有机构账号。

## 十、写在最后

MBA论文是很多人求学路上最难的一道坎。它难的不是写，而是**在有限的时间里，把一个复杂的管理问题讲清楚、分析透彻、给出可执行的方案**。

AI不能替你思考，但它可以帮你：
- 更快地找到相关文献
- 更系统地整理你的思路
- 更高效地把想法变成文字
- 更严格地检查逻辑和格式

**你要做的，是想清楚"我要解决什么问题"。AI帮你做的，是把答案写出来。**
