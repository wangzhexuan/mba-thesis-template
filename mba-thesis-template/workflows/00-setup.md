# Phase 0: 项目初始化

## 目标

完成MBA论文写作项目的初始化配置，包括：
- 确认论文选题和研究方向
- 配置论文基本信息
- 了解学校格式规范
- 建立项目目录结构

## 💡 使用方式

> **你只需要用自然语言告诉AI要做什么，AI会自动调用合适的Skill。**
>
> 例如：你不需要输入 `/paper-lookup "你的研究主题"`，而是直接说：
> "帮我检索关于你的研究主题的文献"
>
> AI会自动选择 `paper-lookup`、`research-lookup` 等Skill完成任务。
> 本工作流中每个步骤都提供了可直接复制的Prompt示例。

## 前置条件

- [ ] 已确定论文选题方向
- [ ] 已了解学校对MBA论文的基本要求
- [ ] 已安装Claude Code

## 核心步骤

### Step 1: 运行初始化向导

```bash
# 进入项目目录
cd mba-thesis-template

# 运行交互式初始化脚本
./init.sh
```

向导将引导你完成以下配置：
1. **基本信息**：论文题目、作者、学校、导师等
2. **研究设计**：研究类型、方法、理论框架
3. **学校规范**：字数要求、查重标准、格式规范

> 💬 **Prompt**（如不想用脚本，可直接告诉AI）：
> ```
> 我要开始写MBA论文了，帮我做项目初始化。
> 题目：{你的题目}
> 作者：{姓名}
> 学校：{学校名}
> 导师：{导师名}
> 研究类型：案例研究 / 问卷调查
> 请帮我生成配置文件和目录结构。
> ```

### Step 2: 确认配置文件

初始化完成后，检查生成的配置文件：

```
config/
├── thesis-config.yaml      # 论文核心配置
├── school-format.yaml      # 学校格式规范
├── research-framework.yaml # 研究框架配置
└── defense-config.yaml     # 答辩准备配置
```

> 💬 **Prompt**（检查配置是否完整）：
> ```
> 请帮我检查 config/ 目录下的配置文件是否完整，
> 列出所有必填项和缺失项。
> ```

### Step 3: 确认目录结构

检查项目目录结构是否完整：

```
项目目录/
├── config/          # 配置文件
├── workflows/       # 工作流文档
├── skills/          # Skills工具包
├── templates/       # 文档模板
├── sources/         # 文献资料
├── drafts/          # 论文草稿
├── reviews/         # 审稿意见
├── output/          # 输出文件
├── README.md        # 项目说明
├── AGENTS.md        # Claude Code指令
└── CLAUDE.md        # 行为准则
```

### Step 4: 熟悉工作流文档

浏览 `workflows/` 目录下的7个工作流文档，了解论文写作全流程：

| 阶段 | 文档 | 核心任务 |
|------|------|---------|
| Phase 0 | 00-setup.md | 项目初始化（本文档） |
| Phase 1 | 01-literature.md | 文献检索与管理 |
| Phase 2 | 02-framework.md | 框架设计 |
| Phase 3 | 03-data-collection.md | 数据收集 |
| Phase 4 | 04-writing.md | 内容生成 |
| Phase 5 | 05-review.md | 质量审校 |
| Phase 6 | 06-defense.md | 答辩准备 |

### Step 5: 了解Skills工具包

> **再次强调：你不需要记住任何Skill命令。** 只要用自然语言描述需求，AI会自动选择合适的工具。

阅读 `skills/skills-guide.md` 了解有哪些能力可用，但实际使用时只需写Prompt。

## 常见问题

### Q1: 初始化脚本运行失败怎么办？

**解决方案**：
1. 检查脚本是否有执行权限：`chmod +x init.sh`
2. 检查bash版本：需要bash 4.0+
3. 手动创建配置文件：参考模板手动填写

### Q2: 不确定选题是否合适怎么办？

> 💬 **Prompt**：
> ```
> 我在考虑以下几个MBA论文选题方向：
> 1. {选题1}
> 2. {选题2}
> 3. {选题3}
> 请帮我分析每个选题的可行性、创新性和数据可得性。
> ```

### Q3: 学校格式规范不确定怎么办？

> 💬 **Prompt**：
> ```
> 我是{学校名}的MBA学生，请帮我查询该校MBA论文的格式规范，
> 包括字体、字号、行距、页边距、字数要求等。
> ```

### Q4: 理论框架不知道怎么选择？

> 💬 **Prompt**：
> ```
> 我的MBA论文研究方向是{方向}，研究对象是{企业}。
> 请帮我推荐适合的理论框架，并说明理由。
> 候选理论：{列出你知道的理论}
> ```
