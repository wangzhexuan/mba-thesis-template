# Phase 3: 数据收集

## 目标

完成论文所需数据的收集、整理和初步分析，包括：
- 设计问卷和访谈提纲
- 执行数据收集
- 整理和清洗数据
- 进行初步分析

## 💡 使用方式

> **你只需要用自然语言告诉AI要做什么，AI会自动调用合适的Skill。**
>
> 本工作流中每个步骤都提供了可直接复制的Prompt示例。
> 你不需要手动输入任何 `/skill` 命令。

## 前置条件

- [ ] 已完成框架设计（Phase 2）
- [ ] 已确定研究方法
- [ ] 已配置 `thesis-config.yaml` 中的方法论部分

## 核心步骤

### Step 1: 设计问卷

> **问卷调查型（survey）**：本步骤是核心环节。问卷直接承载研究假设的测量，需特别注意量表来源、信效度和样本量。详见 `templates/survey-example.md`。

#### 1.1 问卷设计原则

| 原则 | 说明 |
|------|------|
| 目的明确 | 每个题项对应研究问题或假设 |
| 简洁易懂 | 语言清晰，避免歧义 |
| 逻辑合理 | 题目顺序符合逻辑 |
| 长度适中 | 控制在15-20分钟完成 |
| 信效度高 | 经过预测试验证 |

> 💬 **Prompt**（案例研究型问卷）：
> ```
> 请帮我设计一份MBA论文问卷。
> 研究对象：{企业}的{群体}
> 研究主题：{主题}
> 问卷目的：辅助诊断现状，为问题分析提供数据支撑
> 
> 要求：
> 1. 包含导语、基本信息（5-7题）、核心量表（3个维度，每维度4-5题）、开放性问题（2题）
> 2. 核心量表使用李克特5点
> 3. 题项要对应研究问题
> 4. 标注量表来源（如有参考文献）
> 5. 总题数控制在20-25题，填写时间15分钟以内
> ```

> 💬 **Prompt**（问卷调查型问卷）：
> ```
> 请帮我设计一份问卷调查型MBA论文的问卷。
> 研究假设：
> H1: {自变量1}正向影响{因变量}
> H2: {自变量2}正向影响{因变量}
> H3: {中介变量}在{自变量}和{因变量}之间起中介作用
> 
> 要求：
> 1. 每个变量使用成熟量表（标注来源）
> 2. 李克特5点量表
> 3. 每个变量4-6个题项
> 4. 包含1-2个反向题项检验作答质量
> 5. 英文量表需提供中文翻译版本
> ```

#### 1.2 问卷结构设计

**案例研究型问卷**（辅助诊断）：
```
问卷结构：
├── 第一部分：导语（目的、匿名承诺、时间估计）
├── 第二部分：基本信息（5-7题）
├── 第三部分：核心量表（2-4个维度，每维度3-5题）
└── 第四部分：开放性问题（1-2题）
```

**问卷调查型问卷**（核心证据）：
```
问卷结构：
├── 第一部分：导语
├── 第二部分：基本信息（人口统计变量）
├── 第三部分：自变量量表（每个假设对应一个量表）
├── 第四部分：中介/调节变量量表
├── 第五部分：因变量量表
└── 第六部分：开放性问题（可选）
```

### Step 2: 设计访谈提纲

> 💬 **Prompt**：
> ```
> 请帮我设计一份半结构化访谈提纲。
> 访谈对象：{岗位名称}，约{人数}人
> 研究主题：{主题}
> 研究问题：{问题}
> 
> 要求：
> 1. 包含开场白（自我介绍+目的说明+录音许可）
> 2. 热身问题（1-2个背景问题）
> 3. 核心问题（4个主题，每个主题3-4个问题）
> 4. 总结问题（还有什么想补充的）
> 5. 结束语（感谢+后续安排）
> 6. 每个问题标注"追问方向"
> ```

> 💬 **Prompt**（访谈对象选择）：
> ```
> 我需要选择{N}名访谈对象，研究主题是{主题}。
> 请帮我设计访谈对象的选取标准：
> 1. 需要覆盖哪些岗位层级
> 2. 每个层级选几人
> 3. 选取标准（工作年限、部门、职级等）
> 4. 如何确保代表性
> ```

### Step 3: 确定样本

| 研究方法 | 建议样本量 | 说明 |
|---------|-----------|------|
| 问卷调查 | 150-300份 | 有效回收率按60-70%计算 |
| 深度访谈 | 8-15人 | 达到信息饱和 |
| 案例研究 | 1-3个案例 | 深度分析 |

> 💬 **Prompt**：
> ```
> 我的问卷有{N}个题项，研究类型是{类型}。
> 请帮我计算需要发放多少份问卷。
> 考虑因素：题项数、回收率（按{比例}计算）、统计分析要求。
> ```

### Step 4: 执行数据收集

#### 问卷发放注意事项
- 说明调查目的和保密性
- 设置合理的截止时间
- 跟踪回收进度
- 准备备选样本

#### 访谈执行要点
- 访谈前：联系对象、预约时间、准备录音设备
- 访谈中：营造轻松氛围、按提纲引导、灵活追问
- 访谈后：24小时内整理录音、转录文字稿、发送感谢信

### Step 5: 数据整理与清洗

> 💬 **Prompt**（问卷数据清洗）：
> ```
> 请帮我设计问卷数据清洗的检查清单。
> 数据量：{N}份问卷
> 题项数：{N}题
> 包含反向题：{是/否}
> 
> 要求列出：
> 1. 需要剔除的问卷类型和判断标准
> 2. 缺失值处理方法
> 3. 异常值检测方法
> 4. 反向题处理方法
> 5. 清洗后的预期有效样本量
> ```

> 💬 **Prompt**（访谈数据整理）：
> ```
> 请帮我设计访谈数据的编码方案。
> 访谈对象：{N}人
> 访谈主题：{主题}
> 
> 要求：
> 1. 开放式编码：提取关键语句
> 2. 主轴编码：归纳主题类别
> 3. 选择性编码：提炼核心观点
> 4. 输出编码表示例
> ```

### Step 6: 初步数据分析

> 💬 **Prompt**（描述性统计）：
> ```
> 以下是问卷数据的描述性统计结果，请帮我分析：
> {粘贴统计数据}
> 
> 要求：
> 1. 分析样本结构是否合理
> 2. 识别异常值
> 3. 初步判断各维度的得分趋势
> 4. 给出后续分析建议
> ```

> 💬 **Prompt**（信效度分析）：
> ```
> 以下是信度分析结果，请帮我判断是否达标：
> {粘贴Cronbach's α结果}
> 
> 以下是效度分析结果：
> {粘贴KMO和因子分析结果}
> 
> 标准：Cronbach's α > 0.7, KMO > 0.7, 因子载荷 > 0.5
> 未达标的请说明如何改进。
> ```

> 💬 **Prompt**（访谈主题分析）：
> ```
> 以下是{N}位访谈对象的核心回答，请帮我进行主题分析：
> {粘贴访谈记录}
> 
> 要求：
> 1. 提取3-5个核心主题
> 2. 每个主题附代表性语句
> 3. 统计各主题出现频次
> 4. 分析主题间的关联
> ```

## 常见问题

### Q1: 问卷回收率低怎么办？

> 💬 **Prompt**：
> ```
> 我的问卷回收率只有{比例}，太低了。
> 问卷链接：{链接}
> 目标样本量：{N}
> 已回收：{N}
> 请帮我分析原因并给出提升回收率的具体建议。
> ```

### Q2: 访谈对象不配合怎么办？

提前沟通研究价值、承诺保密和匿名、灵活安排时间、请导师协助联系、准备备选对象。

### Q3: 数据质量不高怎么办？

问卷设置测谎题、访谈时注意追问、多渠道数据交叉验证、诚实报告数据局限性。
