# Phase 1: 文献检索与管理

## 目标

完成论文所需文献的检索、筛选、分类和综述撰写，包括：
- 建立系统的文献检索策略
- 收集和整理核心文献
- 撰写高质量的文献综述
- 建立文献管理系统

## 💡 使用方式

> **你只需要用自然语言告诉AI要做什么，AI会自动调用合适的Skill。**
>
> 例如：你不需要输入 `/paper-lookup "你的研究主题"`，而是直接说：
> "帮我检索关于你的研究主题的文献"
>
> AI会自动选择 `paper-lookup`、`research-lookup`、`pdf` 等Skill完成任务。
> 本工作流中每个步骤都提供了可直接复制的Prompt示例。

## 前置条件

- [ ] 已完成项目初始化（Phase 0）
- [ ] 已确定研究方向和关键词
- [ ] 已配置 `thesis-config.yaml` 中的关键词

## 核心步骤

### Step 1: 制定文献检索策略

#### 1.1 确定检索关键词

根据 `thesis-config.yaml` 中的关键词，扩展检索词：

```
核心关键词：
├── 中文关键词（3-5个）
├── 英文关键词（3-5个）
├── 同义词/近义词
└── 上位词/下位词
```

> 💬 **Prompt**：
> ```
> 我的MBA论文题目是：{题目}
> 研究对象：{企业}，行业：{行业}
> 请帮我制定文献检索策略：
> 1. 中文关键词（5-8个，含同义词和上下位词）
> 2. 英文关键词（5-8个）
> 3. 推荐的中英文数据库
> 4. 文献筛选标准（时间、质量、数量）
> ```

#### 1.2 确定检索数据库

**中文数据库**：
- CNKI（中国知网）：核心期刊、硕博论文
- 万方数据：期刊、会议论文
- 维普期刊：中文期刊

**英文数据库**：
- Web of Science：国际核心期刊
- Scopus：综合性学术数据库
- Google Scholar：综合性搜索

#### 1.3 确定文献筛选标准

| 标准 | 要求 |
|------|------|
| 时间范围 | 近5-10年为主 |
| 文献类型 | 期刊论文、学位论文、著作 |
| 质量标准 | 核心期刊、SCI/SSCI |
| 语言分布 | 中文60-70%，英文30-40% |
| 数量要求 | 总计50篇以上 |

### Step 2: 执行文献检索

> 💬 **Prompt**（检索中文文献）：
> ```
> 帮我检索关于"{关键词}"的中文学术文献。
> 要求：
> 1. 优先核心期刊（CSSCI/北大核心）
> 2. 近5年为主
> 3. 返回10篇最相关的文献，包含：作者、年份、标题、期刊、核心观点
> ```

> 💬 **Prompt**（检索英文文献）：
> ```
> Search for academic papers on "{keyword}".
> Requirements:
> 1. From top journals (SCI/SSCI)
> 2. Recent 5 years
> 3. Return 10 most relevant papers with: author, year, title, journal, key findings
> ```

**检索步骤**：
1. 在CNKI中使用高级检索
2. 设置时间、来源、基金等筛选条件
3. 按被引次数排序
4. 导出文献信息（EndNote格式）

#### 文献追溯

> 💬 **Prompt**：
> ```
> 我找到了这篇核心文献：
> {作者}(年份)：{标题}
> 请帮我：
> 1. 找出这篇文献引用了哪些重要文献（向后追溯）
> 2. 找出哪些文献引用了这篇文献（向前追踪）
> 3. 筛选出与我的研究主题{主题}最相关的5篇
> ```

### Step 2.5: 下载和保存文献PDF

> **重要**：检索到文献后，必须下载PDF并保存到项目目录，以便后续阅读和引用。

#### 文献PDF命名规范

```
[作者]_[年份]_[关键词].pdf

示例：
- 张三_2024_供应链管理优化.pdf
- Li_2023_supply_chain_integration.pdf
```

#### 保存到项目目录

```bash
# 经典文献（奠基性文献，5-10%）
mv 经典文献.pdf sources/Classic/

# 核心文献（高质量研究，20-30%）
mv 核心文献.pdf sources/KeyTexts/

# 支撑文献（辅助论证，60-70%）
mv 支撑文献.pdf sources/Supporting/
```

### Step 3: 文献筛选与分类

> 💬 **Prompt**（筛选文献）：
> ```
> 我检索到了以下文献列表，请帮我按以下标准筛选：
> 1. 与我的研究主题{主题}的相关度
> 2. 文献质量（期刊级别、被引次数）
> 3. 时间（近5年优先）
> 
> 请将文献分为三类：经典文献、核心文献、支撑文献。
> 
> 文献列表：
> {粘贴文献列表}
> ```

#### 文献分类体系

| 类别 | 比例 | 说明 | 存放位置 |
|------|------|------|---------|
| 经典文献 | 5-10% | 领域奠基性文献 | `sources/Classic/` |
| 核心文献 | 20-30% | 最新高质量研究 | `sources/KeyTexts/` |
| 支撑文献 | 60-70% | 辅助论证的文献 | `sources/Supporting/` |

#### 建立文献矩阵

> 💬 **Prompt**：
> ```
> 请帮我建立文献矩阵，格式如下：
> | 序号 | 作者 | 年份 | 题目 | 核心观点 | 研究方法 | 与本研究关系 |
> 
> 文献来源：{列出已筛选的核心文献}
> 研究主题：{你的主题}
> ```

### Step 4: 文献精读与笔记

> 💬 **Prompt**（精读文献）：
> ```
> 请帮我精读这篇文献，提取以下信息：
> 
> 文献内容：
> {粘贴PDF内容或提供文件路径}
> 
> 要求提取：
> 1. 研究问题
> 2. 研究方法
> 3. 核心发现（3-5点）
> 4. 理论贡献
> 5. 研究局限
> 6. 与我的研究（{你的主题}）的关系
> 7. 可引用的关键表述
> ```

#### 文献笔记模板

```markdown
## 文献笔记：[文献标题]

**基本信息**
- 作者：
- 年份：
- 期刊/出版社：
- DOI：

**核心内容**
- 研究问题：
- 研究方法：
- 主要发现：
- 管理启示或应用价值：

**与本研究关系**
- 启发点：
- 差异点：
- 可引用内容：

**个人评价**
- 优点：
- 局限：
- 值得借鉴：
```

### Step 5: 撰写文献综述

> 💬 **Prompt**（撰写文献综述）：
> ```
> 请帮我撰写第2章文献综述中的"{主题}"部分。
> 
> 核心文献（已精读）：
> 1. {作者}(年份)：{核心观点}
> 2. {作者}(年份)：{核心观点}
> 3. {作者}(年份)：{核心观点}
> 
> 要求：
> - 按主题组织（非按作者罗列）
> - 批判性分析（比较不同学者的观点）
> - 指出研究空白
> - 字数约{N}字
> - 引用格式：GB/T 7714-2015
> ```

> 💬 **Prompt**（文献评述）：
> ```
> 基于以上文献综述，请帮我撰写文献评述部分（2.4节）。
> 要求：
> 1. 总结现有研究的贡献（3点）
> 2. 指出现有研究的不足（3点）
> 3. 说明本研究的切入点和价值
> ```

#### 文献综述结构

```
2.1 核心概念界定
├── 2.1.1 [概念1]的定义
├── 2.1.2 [概念2]的定义
└── 2.1.3 [概念3]的定义

2.2 理论基础
├── 2.2.1 [理论1]的内涵与发展
├── 2.2.2 [理论2]的核心观点
└── 2.2.3 [理论3]的应用演进

2.3 国内外研究现状
├── 2.3.1 [主题1]研究综述
├── 2.3.2 [主题2]研究综述
└── 2.3.3 [主题3]研究综述

2.4 文献评述
├── 2.4.1 现有研究的贡献
├── 2.4.2 现有研究的不足
└── 2.4.3 本研究的切入点
```

#### 保存输出物

> 💬 **Prompt**（保存文件）：
> ```
> 请将文献综述保存到 drafts/chapter2-literature.md
> ```

## 检查点

### 文献数量检查

- [ ] 总文献数量 ≥ 50篇
- [ ] 中文文献 ≥ 30篇
- [ ] 英文文献 ≥ 20篇
- [ ] 近5年文献 ≥ 20篇
- [ ] 核心期刊文献占比 ≥ 60%

### 文献下载检查

- [ ] 已下载文献PDF ≥ 30篇
- [ ] PDF文件可正常打开
- [ ] 文件命名规范（作者_年份_关键词.pdf）
- [ ] 已按分类保存到sources目录

### 综述质量检查

- [ ] 结构逻辑清晰
- [ ] 批判性分析到位
- [ ] 研究空白已识别
- [ ] 引用格式规范
- [ ] 字数符合要求（5000-8000字）

## 常见问题

### Q1: 文献太多不知道怎么筛选？

> 💬 **Prompt**：
> ```
> 我检索到{N}篇文献，太多了不知道怎么筛选。
> 我的研究主题是{主题}，研究问题是{问题}。
> 请帮我制定筛选标准，并从以下文献中选出最相关的30篇：
> {粘贴文献列表}
> ```

### Q2: 英文文献读不懂怎么办？

> 💬 **Prompt**：
> ```
> 请帮我阅读这篇英文文献，用中文总结核心内容：
> {粘贴英文内容或提供文件路径}
> 要求：提取研究问题、方法、发现、贡献和局限。
> ```

### Q3: 文献综述写成流水账怎么办？

> 💬 **Prompt**：
> ```
> 以下是我写的文献综述初稿，请帮我改进：
> {粘贴内容}
> 
> 问题：写成了流水账，缺乏批判性分析。
> 要求：
> 1. 按主题重新组织
> 2. 加入学者间的对比和评价
> 3. 突出研究空白
> ```

### Q4: 引用格式不统一怎么办？

> 💬 **Prompt**：
> ```
> 请帮我将以下参考文献统一为GB/T 7714-2015格式：
> {粘贴参考文献列表}
> ```
