# Phase 4: 内容生成

## 目标

完成论文全部章节的撰写，包括：
- 按章节顺序完成初稿
- 制作图表和可视化
- 整理参考文献
- 生成论文终稿

## 💡 使用方式

> **你只需要用自然语言告诉AI要做什么，AI会自动调用合适的Skill。**
>
> 本工作流中每个步骤都提供了可直接复制的Prompt示例。
> 你不需要手动输入任何 `/skill` 命令。

## 前置条件

- [ ] 已完成数据收集（Phase 3）
- [ ] 已完成数据分析
- [ ] 已有各章节的初步框架

## 核心步骤

### Step 1: 制定写作计划

#### 1.1 章节写作顺序建议

```
推荐写作顺序：
1. 第三章：现状分析（数据最清晰）
2. 第四章：策略优化（核心贡献）
3. 第二章：文献综述（已有积累）
4. 第一章：绪论（最后完善）
5. 第五章：结论（总结全篇）
```

#### 1.2 写作进度规划

| 章节 | 目标字数 | 计划天数 | 每日字数 |
|------|---------|---------|---------|
| 第一章 绪论 | 3000字 | 3天 | 1000字 |
| 第二章 文献综述 | 6000字 | 6天 | 1000字 |
| 第三章 现状分析 | 8000字 | 8天 | 1000字 |
| 第四章 策略优化 | 8000字 | 8天 | 1000字 |
| 第五章 结论 | 2000字 | 2天 | 1000字 |
| **合计** | **27000字** | **27天** | **1000字** |

> 💬 **Prompt**：
> ```
> 请帮我制定MBA论文写作计划。
> 总字数要求：{字数}字
> 章节数：{5/6}章
> 每天可投入时间：{小时}小时
> 截止日期：{日期}
> 请按章节分配字数和天数。
> ```

### Step 2: 撰写中英文摘要

> **重要**：摘要是论文的"门面"，评审老师首先看的就是摘要。建议在完成所有章节后再撰写摘要。

> 💬 **Prompt**（中文摘要）：
> ```
> 请帮我撰写MBA论文中文摘要（500-800字）。
> 论文题目：{题目}
> 研究对象：{企业}
> 研究方法：{方法}
> 主要发现：
> 1. {发现1}
> 2. {发现2}
> 3. {发现3}
> 主要结论：{结论}
> 
> 要求：
> - 结构：[研究背景]→[研究目的]→[研究方法]→[研究发现]→[研究结论]→[研究价值]
> - 不使用"本文认为"等主观表述
> - 关键词3-5个，用分号分隔
> ```

> 💬 **Prompt**（英文摘要）：
> ```
> 请将以下中文摘要翻译为英文学术摘要（300-500词）：
> {粘贴中文摘要}
> 
> 要求：
> - 与中文摘要内容对应
> - 使用学术英语表达
> - 时态以一般现在时和一般过去时为主
> - Keywords与中文关键词对应
> ```

#### 中文摘要结构模板

```
[研究背景] 随着......的发展，......问题日益突出。
[研究目的] 本文以......为研究对象，运用......理论/方法，研究......。
[研究方法] 本文采用......方法，通过......收集数据，运用......进行分析。
[研究发现] 研究发现：（1）......；（2）......；（3）......
[研究结论] 基于以上分析，本文提出......策略/建议：（1）......；（2）......；（3）......
[研究价值] 本研究对......具有重要的理论和实践意义。
```

### Step 3: 撰写各章节

> **章节结构差异**：案例研究型和问卷调查型在第3-5章有显著差异。详见 `templates/case-study-template.md` 和 `templates/survey-example.md`。

#### 3.1 第一章：绪论

> 💬 **Prompt**：
> ```
> 请帮我撰写MBA论文第1章绪论，约4000字。
> 论文题目：{题目}
> 研究对象：{企业}
> 研究背景：宏观{政策}，行业{趋势}，微观{企业痛点}
> 研究问题：{问题}
> 研究方法：{方法}
> 
> 结构：1.1研究背景及意义 → 1.2研究内容与方法 → 1.3研究边界
> 要求：背景从宏观到微观递进，意义突出实践价值。
> ```

#### 3.2 第二章：文献综述

> 💬 **Prompt**：
> ```
> 请帮我撰写第2章文献综述中的"{主题}"部分，约2000字。
> 核心文献：
> 1. {作者}(年份)：{核心观点}
> 2. {作者}(年份)：{核心观点}
> 3. {作者}(年份)：{核心观点}
> 
> 要求：
> - 按主题组织（非按作者罗列）
> - 批判性分析（比较不同学者观点）
> - 指出研究空白
> - 引用格式：GB/T 7714-2015
> ```

#### 3.3 第三章：现状分析（案例研究型）

> 💬 **Prompt**：
> ```
> 请帮我撰写第3章"{企业}"现状分析，约8000字。
> 
> 企业概况：{基本信息}
> 外部环境数据：{PEST/行业数据}
> 内部环境数据：{SWOT/资源能力}
> 问卷结果：{核心统计数据}
> 访谈发现：{核心主题}
> 
> 结构：
> 3.1 企业概况 → 3.2 外部环境分析 → 3.3 内部环境分析 → 3.4 调研数据分析
> 
> 要求：
> - 数据真实可靠，标注来源
> - 分析有逻辑性，从现象到问题
> - 图表清晰规范（图3-1、表3-1格式）
> - 末尾总结3-5个核心问题
> ```

#### 3.4 第四章：策略优化（案例研究型）

> 💬 **Prompt**：
> ```
> 请帮我撰写第4章优化策略，约8000字。
> 
> 第3章诊断的核心问题：
> 1. {问题1}
> 2. {问题2}
> 3. {问题3}
> 
> 理论框架：{理论1}、{理论2}
> 
> 结构：
> 4.1 优化目标与原则
> 4.2 策略一：{名称}（解决问题1）
> 4.3 策略二：{名称}（解决问题2）
> 4.4 策略三：{名称}（解决问题3）
> 4.5 实施保障
> 
> 要求：
> - 每个策略必须对应第3章的具体问题
> - 措施具体可行，写清动作、对象、条件
> - 每个策略包含：策略框架→具体措施→实施路径
> ```

#### 3.5 第五章：结论

> 💬 **Prompt**：
> ```
> 请帮我撰写第5章结论与展望，约2500字。
> 
> 研究问题：{问题}
> 主要发现：{发现1}、{发现2}、{发现3}
> 主要策略：{策略1}、{策略2}、{策略3}
> 
> 结构：
> 5.1 研究结论（主要结论+结论解读）
> 5.2 研究启示（理论启示+实践启示）
> 5.3 研究局限与展望（局限+未来方向）
> 
> 要求：
> - 结论呼应研究问题
> - 启示有实际价值
> - 局限诚实，不模板化
> ```

### Step 4: 制作图表

> 💬 **Prompt**（统计图表）：
> ```
> 请帮我生成以下数据的统计图表：
> {粘贴数据}
> 
> 图表类型：柱状图/饼图/折线图
> 图表标题：{标题}
> 要求：学术风格，300dpi，标注数据来源。
> ```

> 💬 **Prompt**（研究框架图）：
> ```
> 请帮我生成研究框架图的文字描述（我可以用draw.io绘制）。
> 框架包含：
> 1. 研究问题层
> 2. 理论基础层
> 3. 分析维度层
> 4. 策略设计层
> 5. 实施保障层
> 用箭头标注各层之间的逻辑关系。
> ```

#### 图表规范

```
图表格式要求：
├── 图片格式：PNG/JPG，300dpi以上
├── 图表标题：宋体，小五号
├── 图表编号：按章节编号（图3-1，表3-1）
├── 图表位置：紧跟首次引用的段落后
└── 数据来源：图表下方标注
```

### Step 5: 管理参考文献

> 💬 **Prompt**（格式化参考文献）：
> ```
> 请将以下参考文献统一为GB/T 7714-2015格式：
> {粘贴参考文献列表}
> 
> 要求：
> 1. 期刊论文：[序号] 作者. 题名[J]. 刊名, 年, 卷(期): 页码.
> 2. 专著：[序号] 作者. 书名[M]. 出版地: 出版者, 年: 页码.
> 3. 学位论文：[序号] 作者. 题名[D]. 保存地: 保存单位, 年.
> ```

> 💬 **Prompt**（验证文献信息）：
> ```
> 请帮我验证以下文献的元数据是否准确（DOI、期刊名、年份等）：
> {粘贴文献信息}
> ```

#### 参考文献检查

- [ ] 引用数量满足要求（≥50篇）
- [ ] 中英文比例合理
- [ ] 近5年文献占比≥40%
- [ ] 引用格式统一
- [ ] 正文中引用与文献列表对应

### Step 6: 整合与完善

> 💬 **Prompt**（内容检查）：
> ```
> 请帮我检查论文各章节的逻辑连贯性：
> 第1章问题：{问题}
> 第2章理论：{理论}
> 第3章诊断：{诊断}
> 第4章策略：{策略}
> 第5章结论：{结论}
> 
> 检查：
> 1. 各章是否前后呼应
> 2. 策略是否对应问题
> 3. 结论是否回应研究问题
> 4. 有无逻辑跳跃或矛盾
> ```

> 💬 **Prompt**（生成Word）：
> ```
> 请帮我将论文草稿生成Word文档。
> 草稿目录：drafts/
> 输出目录：output/
> 格式要求：按 config/school-format.yaml 配置
> ```
