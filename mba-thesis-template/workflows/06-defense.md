## 核心步骤

### Step 1: 制作答辩PPT

#### 1.1 PPT结构设计

```
答辩PPT结构（20页，15分钟）：

1. 封面（1页）— 论文题目、作者、导师、学校、日期
2. 目录（1页）— 汇报结构预告
3. 研究背景与问题（3页）— 宏观→行业→微观→问题
4. 文献综述与理论框架（2页）— 核心理论+框架图
5. 研究方法与数据（2页）— 研究设计+样本
6. 核心发现（5页）— 现状分析+问题诊断+数据图表
7. 策略方案（4页）— 优化框架+核心策略+实施路径
8. 研究结论（2页）— 主要结论+贡献
9. 局限性与展望（1页）
10. 致谢（1页）
```

> 💬 **Prompt**（PPT大纲）：
> ```
> 请帮我设计MBA答辩PPT大纲（20页，15分钟）。
> 
> 论文题目：{题目}
> 研究对象：{企业}
> 研究方法：{方法}
> 主要发现：
> 1. {发现1}
> 2. {发现2}
> 3. {发现3}
> 主要策略：
> 1. {策略1}
> 2. {策略2}
> 3. {策略3}
> 主要结论：{结论}
> 
> 要求：
> 1. 每页标题+要点（最多6行）
> 2. 标注建议使用的图表类型
> 3. 标注每页的演讲时间
> 4. 重点突出数据和策略
> ```

#### 1.2 PPT设计原则

| 原则 | 说明 |
|------|------|
| 简洁明了 | 每页文字不超过6行 |
| 重点突出 | 关键数据用醒目颜色 |
| 图表为主 | 用图表代替大段文字 |
| 逻辑清晰 | 页面之间过渡自然 |
| 配色统一 | 使用学校主题色 |

> 💬 **Prompt**（生成PPT）：
> ```
> 请根据以下PPT大纲生成答辩PPT：
> {粘贴大纲}
> 
> 设计要求：
> - 配色：{学校主题色}
> - 字体：标题黑体，正文宋体
> - 每页最多6行文字
> - 关键数据用图表展示
> ```

### Step 2: 准备演讲稿

#### 2.1 演讲时间分配

```
时间分配（15分钟）：
├── 开场白：1分钟
├── 研究背景与问题：2分钟
├── 文献综述与理论框架：1.5分钟
├── 研究方法与数据：1.5分钟
├── 核心发现：4分钟
├── 策略方案：3分钟
├── 研究结论：1.5分钟
└── 结束语：0.5分钟
```

> 💬 **Prompt**（演讲稿）：
> ```
> 请帮我撰写15分钟的答辩演讲稿。
> 
> PPT大纲：
> {粘贴PPT大纲}
> 
> 论文题目：{题目}
> 研究对象：{企业}
> 
> 要求：
> 1. 开场1分钟（自我介绍+论文题目+汇报结构）
> 2. 主体13分钟（按PPT结构，标注每部分时间）
> 3. 结尾1分钟（总结+致谢）
> 4. 口语化表达，避免念稿感
> 5. 标注需要停顿和强调的地方
> 6. 标注需要切换PPT的位置
> ```

#### 2.2 演讲技巧

```
开场技巧：
├── 简洁介绍自己和论文
├── 说明研究的核心问题
└── 预告演讲结构

演讲技巧：
├── 语速适中，不要太快
├── 与评委保持眼神交流
├── 重点内容适当停顿
├── 使用手势辅助表达
└── 控制好时间节奏

结尾技巧：
├── 总结核心结论
├── 感谢评委提问
└── 表达对研究的热爱
```

### Step 3: 准备Q&A

#### 3.1 常见问题分类

| 类别 | 问题示例 | 准备要点 |
|------|---------|---------|
| 澄清类 | "请解释XX概念" | 熟悉核心概念 |
| 方法论 | "为什么选择这种方法" | 说明选择理由 |
| 实践价值 | "方案如何落地" | 准备实施细节 |
| 理论适配 | "为什么用这个理论" | 说明理论与案例的匹配 |
| 应用价值 | "对同类机构有什么参考" | 说明情境边界和可借鉴点 |
| 局限性 | "研究有什么不足" | 诚实承认局限 |
| 细节追问 | "数据来源是什么" | 熟悉论文细节 |

> 💬 **Prompt**（生成Q&A题库）：
> ```
> 请模拟MBA论文答辩评委，根据我的论文内容生成15个可能的问题。
> 
> 论文题目：{题目}
> 研究对象：{企业}
> 研究方法：{方法}
> 主要结论：{结论}
> 研究局限：{局限}
> 
> 要求：
> 1. 问题类型覆盖：澄清类(3个)、方法论(3个)、实践价值(3个)、理论适配(2个)、局限性(2个)、细节追问(2个)
> 2. 每个问题附参考答案要点
> 3. 标注哪些问题可能难以回答，给出应对策略
> ```

> 💬 **Prompt**（模拟答辩）：
> ```
> 请模拟MBA论文答辩，对我进行Q&A练习。
> 
> 论文题目：{题目}
> 研究对象：{企业}
> 主要结论：{结论}
> 
> 请你扮演评委，一次提一个问题，我回答后：
> 1. 评价我的回答（优点和不足）
> 2. 给出改进建议
> 3. 再提下一个问题
> 
> 先从最常见的3个问题开始。
> ```

#### 3.2 回答技巧

```
回答问题技巧：
├── 认真听完问题再回答
├── 不理解可以请评委重复
├── 回答要有条理（第一、第二...）
├── 不知道的诚实说明
├── 不要与评委争论
└── 控制回答时间（1-2分钟）
```

### Step 4: 模拟答辩

#### 4.1 模拟答辩安排

```
模拟答辩计划：
├── 第一次（答辩前3周）— 熟悉流程，发现问题
├── 第二次（答辩前2周）— 改进问题，优化表达
└── 第三次（答辩前1周）— 最终完善，建立信心
```

> 💬 **Prompt**（模拟答辩反馈）：
> ```
> 请对我的模拟答辩进行整体评估。
> 
> 我的演讲内容：
> {粘贴演讲稿}
> 
> 我的回答记录：
> {粘贴Q&A记录}
> 
> 请评估：
> 1. 演讲内容是否完整、逻辑是否清晰
> 2. 时间控制是否合理
> 3. 回答是否准确、有条理
> 4. 哪些地方需要改进
> 5. 整体准备是否充分
> ```

答辩流程：

## 常见问题

### Q1: 答辩紧张怎么办？

充分准备、多次模拟、提前到场、深呼吸放松、把答辩当作交流。

### Q2: 时间控制不好怎么办？

> 💬 **Prompt**：
> ```
> 我的演讲稿超时了，请帮我精简到15分钟以内。
> 当前演讲稿：
> {粘贴演讲稿}
> 要求：保留核心内容，删除冗余部分。
> ```

### Q3: 问题不会回答怎么办？

> 💬 **Prompt**：
> ```
> 评委问了这个问题，我不知道怎么回答：
> "{问题}"
> 
> 我的论文主要研究{主题}，结论是{结论}。
> 请帮我组织一个有条理的回答。
> ```
