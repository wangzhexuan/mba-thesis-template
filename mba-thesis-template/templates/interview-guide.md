# 访谈提纲模板

## 使用说明

本工具提供MBA论文深度访谈的标准提纲结构，可根据研究主题进行调整。

---

## 访谈类型选择

| 访谈类型 | 适用场景 | 时长 | 特点 |
|---------|---------|------|------|
| 结构化访谈 | 标准化信息收集 | 30-45分钟 | 问题固定，便于比较 |
| 半结构化访谈 | 深入探索性研究 | 45-60分钟 | 有提纲，灵活追问 |
| 非结构化访谈 | 开放性探索 | 60-90分钟 | 自由交流，深度探索 |

**MBA论文推荐**：半结构化访谈

---

## 访谈提纲结构

```
访谈提纲组成：
├── 开场（5分钟）
│   ├── 自我介绍
│   ├── 访谈目的说明
│   └── 录音许可
├── 热身问题（5分钟）
│   └── 背景了解
├── 核心问题（30-40分钟）
│   ├── 主题1：问题1-N
│   ├── 主题2：问题1-N
│   └── 主题3：问题1-N
├── 总结问题（5分钟）
│   └── 补充和确认
└── 结束（5分钟）
    ├── 感谢
    └── 后续安排
```

## 访谈提纲模板

### 开场白（5分钟）

```
尊敬的[称呼]：

您好！我是[姓名]，是[学校]MBA研究生。非常感谢您抽出时间接受我的访谈。

我目前正在研究[研究主题]，本次访谈旨在了解[访谈目的]。
访谈大约需要[时长]分钟，您的回答将仅用于学术研究，我们会严格保密。

为了准确记录访谈内容，我希望能够录音，请问您是否同意？
如果您在访谈过程中有任何不适，可以随时要求暂停或终止。

请问您准备好了吗？
```

### 热身问题（5分钟）

```
1. 请您简单介绍一下您的工作经历和目前的岗位职责？

2. 您在[企业/行业]工作多长时间了？

3. 您对[研究主题]的总体印象是什么？
```

### 核心问题

#### 主题1：[主题名称]

```
【背景了解】

4. 请您介绍一下[研究对象]目前的现状？

5. 您认为目前[研究对象]存在哪些主要问题？

【深入探索】

6. 您认为造成这些问题的原因是什么？
   追问：还有其他原因吗？
   追问：这些原因之间有什么关联？

7. 这些问题对[企业/部门/员工]产生了什么影响？
   追问：能举个具体的例子吗？

8. 您认为这些问题中，哪个是最亟需解决的？为什么？
```

#### 主题2：[主题名称]

```
【现状分析】

9. 目前[研究对象]在[方面]的现状如何？

10. 与[标杆/竞争对手]相比，我们的差距在哪里？
    追问：这种差距是如何形成的？

【问题诊断】

11. 您认为在[方面]存在哪些不足？
    追问：能具体描述一下吗？

12. 这些不足对业务发展有什么影响？
    追问：有没有具体的数据或案例可以说明？
```

```

14. 在改进过程中，您认为需要哪些资源支持？
    追问：目前这些资源是否具备？

15. 您对改进后的效果有什么预期？
    追问：如何评估改进效果？
```

再次感谢您的支持！

## 访谈对象设计

### 访谈对象选择原则

1. **代表性**：能够代表研究对象的典型特征
2. **多样性**：不同层级、不同部门、不同背景
3. **信息丰富**：对研究主题有深入了解
4. **配合度**：愿意接受访谈并提供真实信息

### 访谈对象分布建议

| 类别 | 数量 | 选择标准 |
|------|------|---------|
| 高层管理者 | 1-2人 | 战略视角，了解全局 |
| 中层管理者 | 2-3人 | 执行视角，了解管理 |
| 基层管理者 | 2-3人 | 操作视角，了解细节 |
| 普通员工 | 2-3人 | 一线视角，了解实际 |

## 访谈技巧

### 提问技巧

1. **开放式问题**：避免"是/否"问题，使用"为什么"、"如何"
2. **追问技巧**：使用"还有吗"、"能具体说说吗"深入挖掘
3. **中立态度**：不引导，不评价，保持客观
4. **适时沉默**：给受访者思考时间

### 倾听技巧

1. **专注倾听**：保持眼神交流，适时点头
2. **记录要点**：边听边记关键信息
3. **适时回应**：使用"嗯"、"我理解"表示关注
4. **避免打断**：让受访者完整表达

### 追问示例

| 情境 | 追问方式 |
|------|---------|
| 回答太笼统 | "能举个具体的例子吗？" |
| 回答不完整 | "还有其他方面吗？" |
| 回答有矛盾 | "您刚才说...，这怎么理解？" |
| 回答有价值 | "这个观点很有意思，能详细说说吗？" |

## 访谈记录模板

### 访谈基本信息

```
访谈编号：A01
访谈对象：[姓名]
访谈时间：[日期] [时间]
访谈时长：[时长]
访谈地点：[地点]
访谈方式：面对面/电话/视频
记录方式：录音+笔记
```

### 访谈记录格式

```
问题：[问题内容]

回答：[受访者回答内容]

追问：[追问内容]

补充：[补充回答]

[观察记录：受访者的表情、语气、态度等]
```

### 访谈记录示例

```
问题：您认为目前公司绩效管理存在的主要问题是什么？

回答：我觉得最大的问题是指标设置不合理。很多指标都是领导拍脑袋定的，
没有经过科学论证，导致员工觉得不公平。

追问：能举个具体的例子吗？

回答：比如销售部门的KPI，只看销售额，不看利润和客户满意度。
有些销售为了完成指标，低价签单，结果公司亏损。

追问：那您觉得应该如何改进？

回答：应该从多个维度设置指标，不能只看一个数字。可以参考平衡计分卡，
从财务、客户、内部流程、学习成长四个维度来设计。

[观察：受访者谈到这个问题时情绪激动，显然对此深有感触]
```

## 数据分析方法

### 主题分析法步骤

1. **熟悉数据**：反复阅读访谈记录
2. **生成编码**：提取关键语句，进行编码
3. **搜索主题**：将编码归纳为主题
4. **审视主题**：检查主题的合理性
5. **定义主题**：明确每个主题的内涵
6. **撰写报告**：呈现分析结果

## 常见问题

### Q1: 访谈对象不配合怎么办？

**建议**：
1. 提前沟通，说明研究价值
2. 承诺保密和匿名
3. 灵活安排时间地点
4. 请领导协助联系
5. 准备备选对象

### Q2: 访谈跑题怎么办？

**解决方案**：
1. 适时引导回主题
2. 使用"回到刚才的话题..."
3. 记录有价值的信息
4. 控制访谈时间

### Q3: 访谈记录不完整怎么办？

**建议**：
1. 录音是必须的
2. 访谈后24小时内整理
3. 可以再次联系确认
4. 发送记录请受访者确认

### Q4: 如何保证访谈质量？

**方法**：
1. 充分准备访谈提纲
2. 提前演练
3. 保持中立态度
4. 详细记录
5. 及时整理分析
