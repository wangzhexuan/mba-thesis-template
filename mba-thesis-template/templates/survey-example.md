# MBA问卷调查型论文模板

> 本工具提供问卷调查型MBA论文的通用结构和写作指引。与案例研究型的主要区别在于：以大样本量化数据为主要证据来源，强调假设检验和统计分析。

---

## 一、论文基本信息

| 项目 | 内容 |
|------|------|
| 题目 | {研究对象}的{管理问题}研究 —— 基于{样本来源}的调查 |
| 研究类型 | 问卷调查型 |
| 样本量 | {N}份有效问卷 |
| 分析工具 | SPSS / AMOS / Python |

## 三、与案例研究型的关键差异

| 维度 | 案例研究型 | 问卷调查型 |
|------|-----------|-----------|
| 主要证据 | 访谈+文档+观察 | 问卷数据+统计分析 |
| 章节重点 | 第3章现状诊断 | 第3章研究设计 + 第4章数据分析 |
| 理论使用 | 分析框架（PEST/SWOT等） | 研究假设+变量关系 |
| 分析方法 | 主题分析、内容分析 | 回归、SEM、中介/调节 |
| 结论形式 | 问题诊断→策略设计 | 假设验证→管理建议 |
| 字数分布 | 第3-4章最重 | 第3-4章最重 |

## 四、问卷设计要点

### 量表选择

| 量表类型 | 适用场景 | 量级 |
|---------|---------|------|
| 李克特5点 | 态度、感知、满意度 | 1-5分 |
| 李克特7点 | 需要更细致区分 | 1-7分 |
| 频率量表 | 行为频率 | 从不-总是 |

### 问卷结构

```
第一部分：导语（目的、匿名承诺、时间估计）
第二部分：基本信息（5-7题：性别、年龄、学历、工龄、职级等）
第三部分：核心量表（3-5个维度，每维度4-6题）
第四部分：开放性问题（1-2题，可选）
```

### 信效度标准

| 指标 | 标准 | 含义 |
|------|------|------|
| Cronbach's α | > 0.7 | 内部一致性信度 |
| KMO | > 0.7 | 适合做因子分析 |
| 因子载荷 | > 0.5 | 收敛效度 |
| AVE | > 0.5 | 平均提取方差 |
| CR | > 0.7 | 组合信度 |

## 五、样本量要求

| 研究方法 | 建议样本量 | 说明 |
|---------|-----------|------|
| 探索性因子分析 | 5-10倍题项数 | 如20题需100-200份 |
| 验证性因子分析 | ≥ 200 | 结构方程最低要求 |
| 多元回归 | ≥ 50 + 8×自变量数 | 如5个自变量需90+ |
| 结构方程模型 | ≥ 200 | 理想300+ |

有效回收率通常按60-70%计算，发放量 = 目标样本量 ÷ 0.65。
