# 问卷调查模板

## 使用说明

本工具提供MBA论文问卷调查的标准结构和设计方法，可根据研究主题进行调整。

---

## 问卷结构

```
问卷组成：
├── 导语
├── 第一部分：基本信息
├── 第二部分：核心量表
├── 第三部分：开放性问题（可选）
└── 结束语
```

---

## 问卷设计原则

### 题项设计原则

1. **目的明确**：每个题项对应研究问题或假设
2. **简洁易懂**：语言清晰，避免歧义
3. **单一维度**：每个题项只测量一个概念
4. **避免引导**：不暗示"正确"答案
5. **适当反向**：设置反向题项防止惯性作答

### 问卷结构原则

1. **逻辑合理**：题目顺序符合逻辑
2. **先易后难**：简单问题在前，复杂问题在后
3. **先客观后主观**：事实性问题在前，态度性问题在后
4. **长度适中**：控制在15-20分钟完成

### 量表选择建议

| 研究目的 | 推荐量表 | 题项数量 |
|---------|---------|---------|
| 态度测量 | 李克特5点/7点 | 每维度3-5题 |
| 行为频率 | 频率量表 | 每维度2-3题 |
| 满意度 | 满意度量表 | 每维度2-3题 |
| 感知质量 | 语义差异量表 | 每维度3-5题 |

## 信效度检验

### 信度检验

**Cronbach's α系数**：
- α > 0.9：非常好
- 0.8 < α < 0.9：好
- 0.7 < α < 0.8：可接受
- α < 0.7：需要修改

### 效度检验

**内容效度**：
- 专家评审
- 预测试

**结构效度**：
- KMO值 > 0.7
- Bartlett球形检验显著
- 因子载荷 > 0.5

## 常见问题

### Q1: 题项数量多少合适？

**建议**：
- 每个维度3-5个题项
- 总题项15-25个
- 填写时间控制在15分钟内

### Q2: 如何保证问卷质量？

**方法**：
1. 文献中引用的成熟量表
2. 专家评审
3. 预测试（30-50份）
4. 信效度检验

### Q3: 如何提高回收率？

**建议**：
1. 简化问卷
2. 提供小礼品
3. 通过熟人关系发放
4. 多渠道发放
5. 及时提醒
