# 文献综述模板

## 使用说明

本工具提供文献综述的标准结构和写作方法，帮助你系统地梳理和评述相关文献。

---

## 文献综述结构

```
第2章 相关理论与文献综述

2.1 核心概念界定
├── 2.1.1 [概念1]的定义
├── 2.1.2 [概念2]的定义
└── 2.1.3 [概念3]的定义

2.2 理论基础
├── 2.2.1 [理论1]的内涵与发展

└── 2.2.3 [理论3]的应用演进

2.3 国内外研究现状
├── 2.3.1 [主题1]研究综述
├── 2.3.2 [主题2]研究综述
└── 2.3.3 [主题3]研究综述

2.4 文献评述
├── 2.4.1 现有研究的贡献

└── 2.4.3 本研究的切入点
```

## 写作方法

### 2.1 核心概念界定

**写作目的**：明确论文中核心概念的定义和内涵

**写作模板**：

```markdown
2.1.1 [概念1]的定义

[概念1]是指......（给出定义）。

关于[概念1]的定义，学者们从不同角度进行了界定。
Smith（2010）认为[概念1]是......。
Johnson（2015）则将其定义为......。
国内学者张三（2018）认为......。

综合以上观点，本文将[概念1]定义为......。
```

**注意事项**：
- 引用权威定义
- 对比不同观点
- 给出本文的定义
- 注明定义来源

---

### 2.2 理论基础

**写作目的**：介绍支撑研究的理论框架

**写作模板**：

```markdown
2.2.1 [理论1]的内涵与发展

[理论1]是由......提出的，主要用于......。

（1）理论内涵
[理论1]的核心观点包括：第一，......；第二，......；第三，......。

（2）理论发展
[理论1]经历了以下几个发展阶段：
早期阶段（19xx-19xx），......。
发展阶段（19xx-20xx），......。
成熟阶段（20xx至今），......。

（3）在本研究中的应用
本文运用[理论1]分析......，具体体现在：......。
```

**注意事项**：
- 介绍理论起源和发展
- 阐述核心观点
- 说明在本研究中的应用
- 引用经典文献

---

### 2.3 国内外研究现状

**写作目的**：综述相关主题的研究进展

**组织方式**：

#### 方式一：按主题组织（推荐）

```markdown
2.3.1 [主题1]研究综述

（1）国外研究现状

关于[主题1]，国外学者进行了大量研究。

早期研究主要集中在......。Smith（2010）通过研究发现......。
Johnson（2015）进一步指出......。

近年来，研究重点转向......。Williams（2020）采用......方法，
发现......。Brown（2021）的研究表明......。

（2）国内研究现状

国内关于[主题1]的研究起步较晚，但发展迅速。

张三（2018）首次将......引入国内，研究发现......。
李四（2019）通过实证研究指出......。
王五（2020）进一步探讨了......。

（3）研究述评

综合国内外研究，关于[主题1]已取得以下成果：......。
但仍存在以下不足：......。
```

## 文献引用规范

### 引用格式（GB/T 7714-2015）

**期刊论文**：
```
[序号] 作者. 题名[J]. 刊名, 年, 卷(期): 起止页码.
```
示例：[1] 张三, 李四. 企业绩效管理研究[J]. 管理世界, 2020, 45(3): 12-25.

**专著**：
```
[序号] 作者. 书名[M]. 出版地: 出版者, 出版年: 起止页码.
```
示例：[2] Kotler P. Marketing Management[M]. New York: Pearson, 2016: 56-78.

**学位论文**：
```
[序号] 作者. 题名[D]. 保存地: 保存单位, 年份.
```
示例：[3] 王五. 某行业企业管理研究[D]. 北京: 某大学, 2019.

**电子文献**：
```
[序号] 作者. 题名[EB/OL]. (发布日期)[引用日期]. 网址.
```

### 引用注意事项

1. **引用位置**：引用标注在句末标点之前
2. **多篇引用**：用逗号分隔，如[1,3,5]
3. **连续引用**：用短横线连接，如[1-5]
4. **作者引用**：可直接写作者姓名，如"张三（2020）认为......"

## 常见问题

### Q1: 文献太多怎么筛选？

**建议**：
1. 先读综述类文献了解全貌
2. 优先选择核心期刊和高被引文献
3. 与研究主题相关度低的果断舍弃
4. 保持金字塔结构：经典5-10%，核心20-30%，支撑60-70%

### Q2: 文献综述写成流水账怎么办？

**改进方法**：
1. 按主题而非按文献组织
2. 加入对比和评价
3. 使用连接词增强逻辑性
4. 每段有明确的主题句

### Q3: 怎么找到研究空白？

**方法**：
1. 对比不同学者的观点
2. 分析研究方法的局限
3. 识别研究内容的不足
4. 发现新的研究视角

### Q4: 引用格式不统一怎么办？

**解决方案**：
1. 使用Zotero/EndNote等工具管理
2. 统一使用GB/T 7714-2015格式
3. 最终统一检查
4. 使用 `/citation-management` Skills格式化
