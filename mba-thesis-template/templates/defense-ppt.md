# 答辩PPT大纲模板

## 使用说明

本工具提供MBA论文答辩PPT的标准结构，帮助你制作专业的答辩幻灯片。

---

## PPT基本信息

| 项目 | 内容 |
|------|------|
| 总页数 | 15-20页 |
| 答辩时长 | 15分钟 |
| 每页时间 | 约45-60秒 |
| 字体建议 | 标题：微软雅黑/黑体，正文：宋体/微软雅黑 |
| 配色建议 | 使用学校主题色或专业配色 |

## 设计原则

### 内容设计

| 原则 | 说明 |
|------|------|
| 简洁明了 | 每页文字不超过6行 |
| 重点突出 | 关键数据用醒目颜色 |
| 图表为主 | 用图表代替大段文字 |
| 逻辑清晰 | 页面之间过渡自然 |

### 视觉设计

| 元素 | 建议 |
|------|------|
| 配色 | 使用学校主题色或专业配色（2-3种） |
| 字体 | 标题：微软雅黑/黑体，正文：宋体/微软雅黑 |
| 字号 | 标题：28-32号，正文：20-24号 |
| 图片 | 高清、相关、注明来源 |
| 动画 | 简洁、专业、避免花哨 |

### 排版建议

```
页面布局：
├── 标题区域（顶部）
│   ├── 章节标题
│   └── 页码
├── 内容区域（中间）
│   ├── 要点（左侧）
│   └── 图表（右侧）
└── 备注区域（底部）
    └── 数据来源/说明
```

## 图表使用建议

### 推荐图表类型

| 内容 | 推荐图表 | 工具 |
|------|---------|------|
| 数据对比 | 柱状图 | Excel/PPT |
| 占比分析 | 饼图 | Excel/PPT |
| 趋势变化 | 折线图 | Excel/PPT |
| 框架结构 | 流程图 | PPT/Visio |
| 关系分析 | 关系图 | PPT/Visio |
| 时间规划 | 甘特图 | PPT/Visio |

### 图表设计原则

1. **清晰可读**：字体大小合适，颜色对比明显
2. **数据准确**：数据来源明确，标注清楚
3. **风格统一**：图表风格与PPT整体一致
4. **重点突出**：关键数据用醒目颜色标注

## 演讲稿要点

### 开场（1分钟）

```
各位评委老师，下午好！

我是[姓名]，我的论文题目是[论文题目]，
导师是[导师姓名]教授。

下面我将从研究背景、研究方法、核心发现、
策略方案和研究结论五个方面进行汇报。
```

### 主体（13分钟）

- 按PPT结构逐页讲解
- 控制每页时间45-60秒
- 重点内容适当展开
- 数据图表详细说明

### 结尾（1分钟）

```
以上是我的汇报。

本研究的主要结论是：[总结要点]

研究的应用价值在于：[应用价值]

由于时间和能力有限，研究还存在一些不足，
恳请各位老师批评指正。

谢谢！
```

## 常见问题

### Q1: PPT页数太多怎么办？

**建议**：
1. 合并相似内容
2. 删除次要信息
3. 保留核心要点
4. 控制在15-20页

### Q2: 文字太多怎么处理？

**解决方案**：
1. 提炼关键词
2. 使用图表代替
3. 分点陈述
4. 备注栏补充说明

### Q3: 图表不专业怎么办？

**建议**：
1. 使用 `/pptx` Skills生成
2. 参考优秀PPT
3. 使用专业模板
4. 请同学帮忙审阅

### Q4: 时间控制不好怎么办？

**解决方案**：
1. 多次练习计时
2. 准备删减方案
3. 重点内容优先
4. 使用计时器提醒
